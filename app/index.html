Overview

Below are the steps, that are required to set up your website template. Lets get started:

Tools and Installtion

  1. Unzip the downloaded folder. Open the /app folder which contains all the files that are required to host your web server for your website. You have to use FTP for uploading these files.

  2. Below is the folder structure of the downloaded folder:

    • app /
      • css /
        • Thirdparty
      • scss /
      • images /
    • scss /
    • images /
  3. gulpfile.js
  4. package.json
  5. documentation /
    • css /

You should upload only that app folder with all those files that you need for your website. Now under this app/ folder start adding content, images, videos, fonts and other necessary items and modify things as you want. BEST OF LUCK!

Here is what we have under app/ folder:

  • app/css -- Stylesheet files

  • app/fonts -- Icon font file

  • app/images -- Image files

  • app/js -- Javacript files

  • app/video -- Video files

Installation process for running it on local server using Gulp

  1. Node Js:

    You will need gulp for compiling scss to css, and for configuring gulp we need Node Js and node packages. Click here to install Node Js and follow the instructions.

  2. Gulp install:

    After installing Node.js, you need to install Gulp on your system. Run npm install -g gulp command in Terminal/CMD of your system.

  3. Npm install:

    Open Terminal/CMD in the theme folder and run npm install command. This command will install all tools required for this theme.

  4. Running on Browser and compiling Scss:

    To start working, just open Terminal/CMD in the theme folder and run gulp. This will start the working environment for you. Now you can edit the theme in the Code Editor of your choice.
