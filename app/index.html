Overview

Below are the steps, you required to set up your website template. Noe get started:

Tools and Installtion

  1. Unzip the downloaded folder. Open the /HTML folder which contains all the files that are required to host your web server for your website. You have to use FTP for uploading these files.

  2. Below is the folder structure of the downloaded folder:

    • app /
      • css /
        • Thirdparty
      • scss /
      • images /
    • gulpfile.js
    • package.json
    • documentation /
      • css /
      • scss /
      • images /
      • documentation.html /

    You should upload only that app folder with all those files that you need for your website. Now under this app/ folder start adding content, images, videos, fonts and other necessary items and modify things as you want. BEST OF LUCK!

    Here is what we have under app/ folder:

    • app/css -- Stylesheet files

    • app/fonts -- Icon font file

    • app/images -- Image files

    • app/js -- Javacript files

    • app/video -- Video files

    Installation process for running it on local server using Gulp

    1. Node Js:

      We need gulp for compiling scss to css, and for configaring gulp we need Node Js and node packages. Click here to install Node Js and follow the instructions.

    2. Gulp install:

      After installing Node.js, you need to install Gulp on your system. Run npm install -g gulp command in Terminal/CMD of your system.

    3. Npm install:

      Open Terminal/CMD in the theme folder and run npm install command. This command will install all tools required for this theme.

    4. Running on Browser and compiling Sass:

      To start working, just open Terminal/CMD in the theme folder and run gulp. This will start the work environment for you. Now you can edit the theme in the Code Editor of your choice.

Basic HTML Structure

This is the basic html structure of a web page.

<!doctype html>
<html class="no-js" lang="en">
    <head>
        <!-- title -->
        <meta charset="utf-8">
        <!-- description -->
        <meta name="description" content="">
        <!-- keywords -->
        <meta name="keywords" content="">
        <!-- favicon -->
        <!-- style -->
    </head>
    <body>
        <!-- start header -->
        <header>
            <!-- start navigation -->
            <nav class="navbar navbar-default bootsnav">
                ...
            </nav>
            <!-- end navigation -->
        </header>
        <!-- end header -->
        <!-- start sections -->
        <section>
            ...
        </section>
        <!-- end sections -->
        <!-- start footer --> 
        <footer>
            <div class="container">
                ...
            </div>
        </footer>
        <!-- end footer -->
        <!-- javascript libraries -->
    </body>
</html>
